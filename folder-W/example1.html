Введение в HTML

Это начало вашего пути в изучение HTML. Мы рассмотрим основные элементы и структуру HTML-документа.

Урок 1: Основы HTML и структура DOM

Описание

HTML (HyperText Markup Language) — это язык разметки, используемый для создания структуры веб-страниц. DOM (Document Object Model) — это иерархическая структура документа, которая представляет все элементы на веб-странице как объекты.

Основные HTML теги

<!DOCTYPE html>

Описание: Определяет тип документа и версию HTML.

<!DOCTYPE html>

Как выглядит: Не отображается на странице.

<html>

Описание: Корневой элемент HTML-документа.

<html> ... </html>

Как выглядит: Не отображается на странице.

<head>

Описание: Содержит метаинформацию о документе (например, <title>, <meta>).

<head> <title>Page Title</title> </head>

Как выглядит: Не отображается на странице.

<title>

Описание: Определяет заголовок документа, который отображается в заголовке окна браузера.

<title>Page Title</title>

Как выглядит: Отображается в заголовке вкладки браузера.

<meta>

Описание: Предоставляет метаинформацию, такую как кодировка и описание страницы.

<meta charset="UTF-8">

Как выглядит: Не отображается на странице.

<body>

Описание: Содержит все видимые элементы веб-страницы, такие как текст, изображения и другие теги.

<body> ... </body>

Как выглядит: Все содержимое отображается на странице.

<h1> - <h6>

Описание: Заголовки разного уровня от <h1> (самый важный) до <h6> (наименее важный).

<h1>Main Heading</h1> <h2>Subheading</h2>

Как выглядит: Заголовки отображаются как крупный текст, различающийся по размеру в зависимости от уровня.

<p>

Описание: Определяет абзац текста.

<p>This is a paragraph.</p>

Как выглядит: Отображается как блок текста с отступами.

<a>

Описание: Создает гиперссылку на другую страницу или ресурс.

<a href="https://example.com">Visit Example</a>

Как выглядит: Текст ссылки, который можно кликнуть.

Пример HTML документа

<!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <title>Basic HTML Structure</title> </head> <body> <h1>Welcome to HTML</h1> <h2>Introduction</h2> <p>This is a simple HTML document that demonstrates the basic structure and tags used in web development.</p> <a href="https://www.example.com">Learn more about HTML</a> </body> </html>

Объяснение DOM

DOM представляет документ как дерево объектов, где каждый элемент HTML является узлом. Узлы включают элементы, атрибуты, текстовые узлы и комментарии. Это позволяет скриптам (например, JavaScript) динамически изменять содержимое и структуру документа.

Урок 2: Продвинутые теги HTML и работа с DOM

Описание

Этот урок посвящен более сложным тегам HTML, которые расширяют функциональность веб-страниц, а также работе с DOM, позволяющей манипулировать элементами страницы с помощью JavaScript.

Продвинутые HTML теги

<div> и <span>

Описание: <div> используется для блоковой группировки элементов, <span> — для строчной.

<div>This is a block element.</div>
<span>This is an inline element.</span>

<ul>, <ol>, <li>

Описание: Создают списки: упорядоченные (<ol>) и неупорядоченные (<ul>), элементы списка — <li>.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<ol>
  <li>First</li>
  <li>Second</li>
</ol>

<table>, <tr>, <td>, <th>

Описание: Создает таблицу, строки определяются с помощью <tr>, ячейки — <td>, заголовки — <th>.

<table class="table table-bordered">
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>

<form>, <input>, <button>

Описание: Используются для создания форм на веб-страницах. <input> определяет поля ввода, <button> — кнопки.

<form>
  <input type="text" class="form-control mb-2" placeholder="Your name">
  <input type="email" class="form-control mb-2" placeholder="Your email">
  <input type="password" class="form-control mb-2" placeholder="Password">
  <button type="submit" class="btn btn-primary">Submit</button>
</form>

<img>

Описание: Вставляет изображение на веб-страницу.

<img src="image.jpg" alt="Description of image" class="img-fluid">

Пример работы с DOM

DOM позволяет изменять элементы страницы через JavaScript, например, добавлять или удалять элементы, изменять стили и содержимое.

<!DOCTYPE html>
<html>
  <head>
    <title>DOM Manipulation</title>
    <style>
      .highlight { color: red; }
    </style>
  </head>
  <body>
    <h1 id="title">Hello, World!</h1>
    <button onclick="changeText()" class="btn btn-success">Change Text</button>

    <script>
      function changeText() {
        const title = document.getElementById('title');
        title.textContent = 'Text Changed!';
        title.classList.add('highlight');
    }
    </script>
  </body>
</html>

Объяснение примера

В этом примере на странице находится заголовок и кнопка. При нажатии на кнопку срабатывает функция changeText, которая изменяет текст заголовка и добавляет к нему CSS-класс highlight, изменяющий цвет текста на красный.
