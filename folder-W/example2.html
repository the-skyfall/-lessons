Основные стили

Этот раздел покажет основные стили для HTML-элементов.

<div class="basic-styles">
        <h1>Заголовок</h1>
        <p>Это абзац текста.</p>
        <a href="#">Ссылка</a>
    </div>
<style>
                    /* Стили для заголовков внутри блока с классом basic-styles */
                    .basic-styles h1 {
                        color: #333; /* Цвет заголовка */
                    }
    
                    /* Стили для абзацев внутри блока с классом basic-styles */
                    .basic-styles p {
                        color: #666; /* Цвет текста абзаца */
                    }
    
                    /* Стили для ссылок внутри блока с классом basic-styles */
                    .basic-styles a {
                        color: #007bff; /* Цвет ссылки */
                        text-decoration: none; /* Убираем подчеркивание у ссылки */
                    }
    
                    /* Стили для ссылок при наведении */
                    .basic-styles a:hover {
                        text-decoration: underline; /* Подчеркивание при наведении */
                    }
                    </style>

Анимация кнопки

Пример использования CSS-анимации с псевдоэлементами:

<div class="animation-example">
        <button>Анимация</button>
    </div>
<style>
                    /* Стили для кнопки внутри блока с классом animation-example */
                    .animation-example button {
                        position: relative; /* Позиционирование относительно родителя */
                        background-color: #007bff; /* Цвет фона кнопки */
                        color: #fff; /* Цвет текста кнопки */
                        border: none; /* Убираем стандартные границы кнопки */
                        padding: 10px 20px; /* Внутренние отступы кнопки */
                        border-radius: 5px; /* Скругление углов кнопки */
                        overflow: hidden; /* Скрытие содержимого, выходящего за границы кнопки */
                    }
    
                    /* Псевдоэлемент ::after для кнопки */
                    .animation-example button::after {
                        content: ''; /* Пустое содержимое для псевдоэлемента */
                        position: absolute; /* Абсолютное позиционирование */
                        top: 0; /* Привязка к верхнему краю */
                        left: 0; /* Привязка к левому краю */
                        width: 0; /* Начальная ширина псевдоэлемента */
                        height: 100%; /* Высота псевдоэлемента равна высоте кнопки */
                        background-color: rgba(255, 255, 255, 0.3); /* Полупрозрачный цвет фона */
                        transition: width 0.4s ease; /* Переход ширины с анимацией */
                    }
    
                    /* Изменение ширины псевдоэлемента при наведении */
                    .animation-example button:hover::after {
                        width: 100%; /* Заполнение всей ширины кнопки при наведении */
                    }
                    </style>

Псевдоэлементы

Примеры использования псевдоэлементов ::before и ::after:

<div class="pseudo-elements">
        Текст с псевдоэлементами
    </div>
    
    <style>
        /* Псевдоэлемент ::before добавляет текст перед содержимым блока */
        .pseudo-elements::before {
            content: "Привет! "; /* Текст перед содержимым */
            color: #007bff; /* Цвет текста псевдоэлемента */
        }
    
        /* Псевдоэлемент ::after добавляет текст после содержимого блока */
        .pseudo-elements::after {
            content: " - Спасибо за внимание!"; /* Текст после содержимого */
            color: #28a745; /* Цвет текста псевдоэлемента */
        }
        </style>

Медиа-запросы

Пример использования медиа-запросов для изменения стиля на разных экранах:

<div class="media-query-example">
        Этот блок меняет цвет фона на мобильных устройствах.
    </div>
    
    <style>
        /* Основные стили для блока */
        .media-query-example {
            background-color: #007bff; /* Цвет фона */
            color: #fff; /* Цвет текста */
            padding: 10px; /* Внутренние отступы */
        }
    
        /* Медиа-запрос для экранов шириной до 600px */
        @media (max-width: 600px) {
            .media-query-example {
                background-color: #28a745; /* Изменение цвета фона для мобильных устройств */
            }
        }
        </style>
Этот блок меняет цвет фона на мобильных устройствах.

Анимации с ключевыми кадрами

Примеры анимаций с использованием ключевых кадров:

<div class="keyframes-example"></div>
    
    <style>
        /* Анимация вращения элемента */
        .keyframes-example {
            width: 100px; /* Ширина элемента */
            height: 100px; /* Высота элемента */
            background-color: #007bff; /* Цвет фона элемента */
            animation: spin 2s linear infinite; /* Применение анимации вращения */
        }
    
        /* Определение ключевых кадров для вращения */
        @keyframes spin {
            0% { transform: rotate(0deg); } /* Начальное положение */
            100% { transform: rotate(360deg); } /* Конечное положение */
        }
        </style>
<div class="keyframes-example2"></div>
    
    <style>
        /* Анимация подпрыгивания элемента */
        .keyframes-example2 {
            width: 100px; /* Ширина элемента */
            height: 100px; /* Высота элемента */
            background-color: #007bff; /* Цвет фона элемента */
            animation: bounce 1.5s ease-in-out infinite; /* Применение анимации подпрыгивания */
        }
    
        /* Определение ключевых кадров для подпрыгивания */
        @keyframes bounce {
            0%, 20%, 50%, 80%, 100% {
                transform: translateY(0); /* Начальное и конечное положение */
            }
            40% {
                transform: translateY(-30px); /* Положение на пике прыжка */
            }
            60% {
                transform: translateY(-15px); /* Положение после пика прыжка */
            }
        }
        </style>

Градиенты и обрезка текста

Использование градиентов и обрезка текста с помощью фона:

<div class="gradient-background"></div>
    
    <div class="clip-text">
        Градиентный текст
    </div>
    
    <style>
        /* Блок с градиентным фоном */
        .gradient-background {
            width: 200px; /* Ширина блока */
            height: 200px; /* Высота блока */
            background: linear-gradient(45deg, #ff6f61, #6b5b95); /* Градиентный фон */
            border-radius: 10px; /* Скругление углов */
        }
    
        /* Блок с обрезкой текста */
        .clip-text {
            background: linear-gradient(45deg, #ff6f61, #6b5b95); /* Градиентный фон текста */
            -webkit-background-clip: text; /* Обрезка фона по форме текста */
            -webkit-text-fill-color: transparent; /* Сделать текст прозрачным, чтобы показать градиент */
            font-size: 40px; /* Размер шрифта */
            font-weight: bold; /* Жирность шрифта */
        }
        </style>

            padding: 10px 20px; /* Внутренние отступы */
